This code project is meant to convert given amount in one currency to other.
Here, various currencies and Conversion rates between them are configured through yaml.
To start program App class needs to be executed.
It will prompt for line input "<BASE_CUR> <Input Amount> in <TARGET_CUR>".
Output will display the converted amount in following format "<BASE CUR> <Input Amount> = <TARGET_CUR> <Calculated Amount>".
